## 📋 작성된 테스트 케이스

### 1. **기본 동작** (3개 테스트)
- ✅ 시작일 < 종료일: 유효
- ✅ 시작일 = 종료일: 유효  
- ❌ 시작일 > 종료일: 무효

### 2. **문자열 입력 처리** (4개 테스트)
- ✅ ISO 8601 형식 문자열 처리
- ✅ 문자열과 Date 객체 혼합 사용
- ❌ 잘못된 날짜 형식
- ❌ 빈 문자열

### 3. **에러 처리** (7개 테스트)
- ❌ null/undefined 입력값들
- ❌ Invalid Date 객체

### 4. **엣지 케이스** (10개 테스트)
- ✅ 같은 날짜의 다른 시간
- ✅ 1일 차이
- ⚠️ 100년 이상 차이 (경고 메시지)
- ✅ 윤년 2월 29일
- ✅ 타임존 차이

### 5. **경계값 테스트** (4개 테스트)
- ✅ 1900년 이전 날짜
- ✅ 미래 날짜
- ✅ 연도/월 경계

### 6. **반환 타입 검증** (3개 테스트)
- 유효: `{ isValid: true }`
- 무효: `{ isValid: false, error: string }`
- 경고: `{ isValid: true, warning: string }`
